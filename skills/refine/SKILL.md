# Refine

Interview the user relentlessly about a plan or design until reaching shared understanding. Resolve each branch of the decision tree and provide recommended answers as the conversation progresses.

## When to Use

Use this skill when the user wants to stress-test a plan, refine, analyze, critique, get questioned on their design, brainstorm, or mentions "challenge me or my idea".

## Steps

1. Interview the user relentlessly about every aspect of this plan until we reach a shared understanding.
2. Walk down each branch of the design tree resolving dependencies between decisions one by one.
3. If a question can be answered by exploring the codebase, explore the codebase instead. Leverage subagents for exploring codebase where possible.
4. For each question, provide your recommended answer.
